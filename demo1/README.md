Tutorial 1: Node.js + Express 4 + MySQL
===

### To install:

```sh
sudo npm install
```

### Database:
Import the database structure from this file: database.json

### To execute:

```sh
sudo node server.js
```

### For more information:

http://www.pabloanaya.com/blog/building-api-node-js-express-4-mysql
